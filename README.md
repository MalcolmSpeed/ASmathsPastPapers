# Edexcel AS Level Mathematics Past Papers

A collection of Edexcel AS Level Mathematics (specification code **8MA0**) past papers, organised by year and paper number.

## Qualification Overview

**Edexcel AS Level Mathematics (8MA0)** — Pearson Edexcel, first examined May/June 2018.

| Paper | Title | Duration |
|-------|-------|----------|
| Paper 1 (8MA0/01) | Pure Mathematics | 2 hours |
| Paper 2 (8MA0/02) | Statistics & Mechanics | 1 hour 15 minutes |

> **Note:** This repository covers strictly **AS Level (Year 1 / AS only)** papers — not A Level Year 2 or full A Level papers. A Level Year 2 papers use specification code **9MA0**.

## Contributing

If you have access to freely-redistributable copies of Edexcel AS Level Mathematics (8MA0) past papers (e.g. from Pearson with permission), please place them in the appropriate year/paper folder following the naming convention:

```
YYYY/Paper-1/question-paper.pdf
YYYY/Paper-1/mark-scheme.pdf
YYYY/Paper-2/question-paper.pdf
YYYY/Paper-2/mark-scheme.pdf
```

If the year or paper number is unknown, place the file in the `unsorted/` folder.

> **Important:** Only include files that you have the right to redistribute. Uploading copyrighted Pearson Edexcel exam papers without permission is not permitted.
